# Vitale Arbeitskultur Thunderbird Theme.

Offizielles Thunderbird-Theme im Corporate Design von Vitale Arbeitskultur.

## Voraussetzungen

- Mozilla Thunderbird 78.0 oder neuer

## Installation

### Methode 1: Aus Datei installieren

1. Die Datei `va-thunderbird-theme.xpi` herunterladen
2. Thunderbird starten
3. Zum Add-ons-Manager navigieren: **Extras** > **Add-ons und Themes** (oder `Strg+Shift+A`)
4. Auf das Zahnrad-Symbol klicken und **Add-on aus Datei installieren** auswählen
5. Die heruntergeladene `.xpi`-Datei auswählen
6. Installation bestätigen

### Methode 2: Drag & Drop

1. Die Datei `va-thunderbird-theme.xpi` herunterladen
2. Thunderbird starten
3. Die `.xpi`-Datei direkt in das Thunderbird-Fenster ziehen
4. Installation bestätigen

## Farbschema

Das Theme verwendet die offiziellen Markenfarben:

| Farbe | Hex-Code | Verwendung |
|-------|----------|------------|
| Dunkel-Petrol | `#21474d` | Haupthintergrund |
| Gelb | `#efb708` | Akzente, Hervorhebungen |
| Helleres Petrol | `#1a3a3f` | Eingabefelder, Sidebar |

## Deinstallation

1. **Extras** > **Add-ons und Themes** öffnen
2. Zum Bereich **Themes** navigieren
3. Bei „Vitale Arbeitskultur" auf die drei Punkte klicken
4. **Entfernen** auswählen
