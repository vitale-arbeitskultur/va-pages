Interne Kommunikation

Wir nutzen Matrix/Element als sichere, selbst-gehostete Kommunikationsplattform für Chat und Videoanrufe zwischen allen Komplizen.

Unsere Raumstruktur

Die Struktur unserer Matrix-Räume spiegelt unsere soziokratische Organisation wider. So findest du dich schnell zurecht:

Haupträume

#alle-komplizen Hauptkanal für den täglichen Austausch
#ankündigungen Wichtige Mitteilungen (nur Admins können posten)
#random Off-topic, Socializing, Lustiges

Kreise (Space)

Jeder Kreis hat seinen eigenen Raum für die Governance-Arbeit:

#kreis-od Organisationsentwicklung
#kreis-admin Administration
#kreis-coaching Coaching
#kreis-digi Digitalisierung
#spannungen Spannungsverarbeitung (kreisübergreifend)

Projekte (Space)

Für aktive Aufträge werden Projekträume erstellt: #projekt-[name]
Du wirst zu den Projekträumen eingeladen, an denen du mitarbeitest.

Element einrichten

Element ist der Client für Matrix. Du kannst Element auf allen Geräten nutzen.

Desktop (empfohlen)

  1. Lade Element herunter: element.io/download
  2. Öffne Element und wähle "Anmelden"
  3. Gib die Homeserver-URL ein (vom Admin)
  4. Melde dich mit deinen Zugangsdaten an

Mobil

  1. Lade "Element" aus dem App Store / Play Store
  2. Tippe auf "Anmelden"
  3. Wähle "Anderer Homeserver" und gib die URL ein
  4. Melde dich mit deinen Zugangsdaten an

Web

Du kannst Element auch direkt im Browser nutzen. Öffne einfach die Homeserver-URL in deinem Browser.

Videoanrufe

Element Call ist direkt in Matrix integriert. Du brauchst keine zusätzliche Software.

So startest du einen Anruf:

  1. Öffne einen beliebigen Raum
  2. Klicke oben rechts auf das Video-Symbol
  3. Andere Teilnehmer können dem Anruf beitreten

Funktioniert im Browser, Desktop und mobil.

Tipps für die Nutzung

Benachrichtigungen

Stelle Benachrichtigungen so ein, dass du wichtige Nachrichten mitbekommst, aber nicht ständig abgelenkt wirst. In den Raumeinstellungen kannst du die Benachrichtigungsstufe anpassen.

Schlüsselsicherung

Aktiviere die Schlüsselsicherung, damit du deine verschlüsselten Nachrichten auf allen Geräten lesen kannst. Element führt dich beim ersten Start durch die Einrichtung.

Threads

Nutze Threads für längere Diskussionen zu einem Thema. So bleibt der Hauptkanal übersichtlich. Klicke auf eine Nachricht und wähle "In Thread antworten".
